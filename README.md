# Catalog App

## Tech Stack

### Frontend

-   React
-   Chakra UI
-   TailwindCSS

#### Run

```bash
npm run dev
```

### Backend

-   Node.js
    -   Express
    -   jsonwebtoken
    -   bcrypt
    -   dotenv
    -   mongoose
    -   cookie-parser
    -   nodemon (dev)
-   MongoDB

#### Run

```bash
nodemon
```
